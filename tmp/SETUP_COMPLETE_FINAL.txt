Create file: /ros2_ws/src/sensors/lidar/lidar/lidar_processor.py

with content:

  import rclpy
  from sensor_msgs.msg import PointCloud2
  from rclpy.node import Node
  import sensor_msgs.point_cloud2 as pc2
  
  class LidarProcessor(Node):
      def __init__(self):
          super().__init__('lidar_processor')
          self.subscription = self.create_subscription(
              PointCloud2,
              '/cloud',
              self.lidar_callback,
              10)
      
      def lidar_callback(self, msg):
          # Access point cloud
          points = pc2.read_points(msg, skip_nans=True)
          for x, y, z, intensity in points:
              # Process each point
              if z < 0.5:  # Ground points
                  continue
              # Your processing here
              pass
          self.get_logger().info(f'Processed {len(points)} points')
  
  def main(args=None):
      rclpy.init(args=args)
      processor = LidarProcessor()
      rclpy.spin(processor)
      rclpy.shutdown()
  
  if __name__ == '__main__':
      main()

================================================================================
TROUBLESHOOTING
================================================================================

LIDAR CONNECTION FAILED:
  1. Verify IP: ping 10.10.10.211
  2. Check network cable
  3. Try default IP: lidar_hostname:=192.168.0.1
  4. Check firewall settings

CAMERA NOT FOUND:
  1. List USB devices: lsusb | grep Intel (RealSense)
  2. Try device 1: camera_device:=1
  3. Check USB cable connection

IMU SERIAL ERROR:
  1. List serial ports: ls -la /dev/ttyUSB*
  2. Check baud rate (default 115200)
  3. Try different port: imu_port:=/dev/ttyUSB0

GPS TIMEOUT:
  1. May need 30+ seconds for first fix
  2. Ensure antenna is outdoors with sky view
  3. Check serial connection

NO /CLOUD TOPIC:
  1. Verify sensors launched: ros2 topic list
  2. Check lidar is reachable: ping 10.10.10.211
  3. View errors: ros2 launch ... (check stderr output)
